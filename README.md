# thinone

一个使用thinkjs+typescript ,加上B-Jui后端ui框架，加上mui移动端框架的开发模板。

## screen image
![图片](http://blueswu.gitee.io/myhome/thinonep1.png)
## Install dependencies

```
npm install
```

## Start server

```
npm start
```

## Deploy with pm2

Use pm2 to deploy app on production enviroment.

```
pm2 startOrReload pm2.json
```
